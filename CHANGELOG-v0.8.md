# v0.8 — Website Capture

- Added Website / Article selection to Chrome extension and Web App Add link.
- Added automatic page-type recommendation with user override.
- Website saves skip readable-text extraction.
- Chrome extension captures the current visible viewport.
- Web Add link attempts a local headless-Chrome screenshot.
- Screenshots are saved as WebP, max 1440 × 1000, target under 500 KB, hard limit 2 MB.
- Website screenshot opens the original site directly from Library and Reader.
- Website screenshots support manual deletion and local media cleanup.
- Same URL can be saved once as a Website and once as an Article.
