# Revisit v1.0

Stable local-first release.

## Included fixes and polish

- Renamed the product from Later to **Revisit** and standardized the tagline: **Save now. Return with intention.**

- Includes the Website capture hotfix: Website saves send only metadata and the viewport screenshot, avoiding `400` validation failures on long sites.
- Article captures avoid redundant legacy text payloads.
- Extension validation errors surface the failing field.
- Highlights list cards now preserve intentional inner spacing around source labels, quoted text, notes, and actions; text no longer sits against the card border.

## Stable baseline

- Web App and Chrome extension local workflow
- Article / Website save type switch
- Website viewport screenshots with local WebP compression
- WeChat text and inline-image extraction
- X Smart Capture for links, images, and videos without model use
- Reader, Highlights, Notes, title editing, status filters, and local media cleanup
